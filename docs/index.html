Why InboxDNA

Your emails deserve better than the cloud

Cloud-based email cleaners require full access to your mailbox on their servers. InboxDNA runs entirely on your computer.

🔒

100% local processing

Your emails are analyzed on your machine. Nothing is sent to any server, ever. No cloud infrastructure, no API calls, no third-party access.

💰

No subscriptions, no upsells

Free forever. No freemium limits, no premium tiers, no recurring charges. Install it once and use it whenever you need it.

🛠

Open source and auditable

Every line of code is on GitHub under the Apache 2.0 license. Inspect, modify, and contribute. Trust is earned, not demanded.

Features

Everything you need to take back your inbox

13 tools for analyzing, cleaning, and maintaining your Gmail — all running locally on your machine.

🗑

Bulk Cleanup

Scan your inbox, see top senders ranked by volume, and delete, block, or unsubscribe from them in bulk.

⚡

Triage Mode

Rapid one-click review per sender. Keep, archive, delete, block, or unsubscribe with a single action.

🏆

Inbox Hygiene Score

A 0-100 composite score measuring inbox health, with streaks and badges to keep you motivated.

📊

Storage Cost Visualizer

Interactive treemap showing storage usage by sender, with Google One cost estimates for your inbox size.

📡

Subscription Decay Radar

Spot subscriptions you've stopped reading. See open rates and engagement trends over time.

🕔

Quiet Hours Heatmap

Visualize when emails arrive throughout the week. Discover who's emailing you at 3am.

💬

Reply Debt Tracker

Find unanswered emails that may need a response. Never miss an important conversation again.

⏰

Inbox Time Machine

Track how your inbox has grown and changed over time. See composition trends and growth patterns.

👻

Ghost Rules

Auto-suggest Gmail filters based on your triage patterns. Turn manual decisions into automated rules.

🧬

Email DNA Cards

Deep profile for any sender: email frequency, tracking pixels, unsubscribe reliability, and dark patterns.

👁

Privacy Audit

Scan for tracking pixels, sensitive data exposure, and known tracker domains across your inbox.

🌑

Dark Mode

Automatic theme switching that follows your OS preference, or toggle manually between light and dark.

⌨

Keyboard Shortcuts

Navigate with J/K, select with X, archive with E, and press ? for the full shortcut reference.

Install

Up and running in under a minute

Choose the install method that works best for you. No account needed.

Recommended

📦 pip install

Install from PyPI. Requires Python 3.10 or later.

pip install inboxdna inboxdna

💾 Standalone binary

Download a pre-built executable. No Python required.

# Download from GitHub Releases: github.com/blaiseratcliffe/inboxdna/releases # Then run: ./inboxdna # macOS / Linux inboxdna.exe # Windows

💻 From source

Clone the repository and install in development mode.

git clone https://github.com/blaiseratcliffe/inboxdna.git cd inboxdna python -m venv venv source venv/bin/activate # Windows: venv\Scripts\activate pip install -e . inboxdna

Privacy

Built for trust, not engagement

InboxDNA was designed from day one with a simple principle: your emails are yours alone.

🔒

Runs on your machine only

All email analysis happens locally. Your Gmail data is fetched directly to your computer and never relayed through external servers.

🚫

Zero telemetry

No analytics, no crash reports, no usage tracking. InboxDNA makes zero network requests beyond your Gmail connection.

🔎

Fully auditable

The entire codebase is open source under Apache 2.0. Read every line, verify every network call, build from source.

🔓

Revoke anytime

InboxDNA uses standard OAuth. Remove access from your Google account settings at any time and all permissions are revoked instantly.

📧 Gmail API
↓
💻 Your computer (InboxDNA)
↓
💾 Local database only
🚫 No data sent to external servers

InboxDNA connects directly to Gmail's API using your own OAuth credentials. All data stays in a local SQLite database on your machine.
